# Example 01: Basic PostgreSQL Module

This example demonstrates the simplest possible Keelo setup: a single module with custom variable values.

## Files
- `project.yaml`: The project configuration defining which modules to use and their values.
- `modules/postgres/`: A local module that defines a PostgreSQL service.
- `generated/docker-compose.yaml`: The final output generated by Keelo.

## How to run
From the root of the Keelo repository, run:
```bash
keelo render -c examples/01-basic-postgres/project.yaml -o examples/01-basic-postgres/generated/docker-compose.yaml
```

*Note: In this example, we ran the command from within the directory for simplicity.*
